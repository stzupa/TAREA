Lanza los dados

Juega contra la máquina! Debes implementar un lanzador de dados (número aleatorio del 1 – 6) e implementar un botón “lanzar dado” que te entregue el número que te ha salido, luego comparar con el número del dado que lanzó la máquina para luego mostrar quien logró sacar el número mayor. (Considerar empate).

                public class ViewDados {

                        inicio();

                    public void inicio(){
                        JFrame marco_dados = new JFrame();
                        marco_dados.setSize(300, 400);
                        marco_dados.setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);
                        marco_dados.setLocationRelativeTo(null);
                    
                        JPanel panel_dados = new JPanel();
                        marco_dados.add(panel_dados);
                        posicionar(panel_dados);
                        marco_dados.setVisible(true);
                    }

                        text3.setBounds(50, 280, 30, 30);
                        text4.setBounds(200, 250, 60, 30);
                        text5.setBounds(220, 280, 30, 30);
                        text1.setFont(new java.awt.Font("Tahoma", 0, 18));
                    
                        boton_lanzar.addActionListener(new ActionListener(){

                                text1.setText("");

                                text3.setText("");

                                if(n==x){
